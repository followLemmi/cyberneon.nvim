# Screenshots

Place your screenshots here to showcase the cyberdyne-port theme.

## Suggested Screenshots

1. **Code example** - Show syntax highlighting with a code sample
2. **Dashboard** - Show the welcome screen with the theme
3. **File tree** - Neo-tree or NvimTree with the theme
4. **Telescope** - Fuzzy finder with theme colors
5. **LSP diagnostics** - Show error/warning highlighting
6. **Git integration** - GitSigns or diff view

## Recommended Format

- PNG format
- At least 1920x1080 resolution
- Show actual code with the theme applied
- Include popular plugins in action

## Example Screenshot Command

```bash
# Take a screenshot of your Neovim with the theme
# On Linux (using ImageMagick):
import screenshot-code.png

# On macOS:
screencapture -i screenshot-code.png
```

After adding screenshots, reference them in the main README.md like this:

```markdown
## Screenshots

![Code Example](screenshots/code-example.png)
![Dashboard](screenshots/dashboard.png)
```
